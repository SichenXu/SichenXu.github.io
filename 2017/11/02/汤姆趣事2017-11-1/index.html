2017-11-02 汤姆日记

发表于 2017-11-02 | 分类于 Diary

晚上搂着汤姆一起睡着，是一天中最幸福最安心的时刻。

写在前面

这两天配置了一个自己的博客，以后可以在这记录点东西了，开心！正好是11月初，来张图片应个景～

吃桔子

昨天晚上我们吃完晚饭以后，我看到桌上只有一个桔子了，故意跟Tom说妈妈给你剥桔子吃。Tom说我们分享吧，然后S先生说那爸爸给你剥，我们两分享。我故意说还是妈妈来跟你分享吧。Tom说我来分给大家，大家都有份。我剥了皮递给他，他用胖胖的小手剥开一瓣一瓣的，一边剥一边嘴上鼓囊着说：“这个给妈妈”，递给我后又剥了一片递给S先生：“这个给爸爸。“接着又剥一片，说道：“这个是我的”。然后就这么用他小小的手把桔子一片一片的分完了。

假扮钢铁侠

昨天跟Tom涂了色以后，他又找出了新玩法。他把蜡笔的笔帽套在手指上，在满屋子里瞎跑，假扮自己是钢铁侠。然后冲到我面前指着我说“我要把你变成红色”，一会说“我要把你变成蓝色”。喝晚安奶的时候，我说把笔帽拿下来把要不不好拿奶瓶了，他不干坚持笨拙的拿着奶瓶。我给他讲《好饿好饿的毛毛虫》的故事，顺便教他”Monday“，”Tuesday“…一边教一边用手指比划1，2，3…教他。他嫌我笔画3的时候手势不对，手指没有伸直。他先是笨拙的用钢铁侠的手放下奶瓶，一边说“你手不对”，然后把笔套一个一个拿下来，我看着觉得好麻烦就说别拿了别拿了，他坚持的说不行不行，还嘟囔着说“我要来教你”，然后认真的比划着教了我后，又一个一个认真的套了回去。有时候看他做事，觉得小孩子做事反而有一种说不出的较真。他坚持的事情他一定要做，不像我们觉得这个可能会麻烦吧，那个会怎么怎么样了吧，他们可没有那么多顾虑，羡慕！

最喜欢谁的玩具？

昨天幼儿园每个小朋友都要带一样最喜欢的玩具去幼儿园和大家分享。Tom带了一个沙滩卡车。睡前我跟他聊天，问他这个小朋友带的什么那个带的什么，他们组的他都能一一说出，有的是坦克啦，有的是小火车啦，有的是变形恐龙啦，其他组的小朋友有的他不记得的他就说：“我不记得了，哈哈”我问他：“那你最喜欢哪个？”他说：“我的沙滩卡车”，我又问他，那如果想让妈妈给你买一样新的玩具，你想要哪个呢？“他说：”那就坦克吧，哈哈！“我问他问什么，他说：”因为坦克可以打败很多敌人呀！“
